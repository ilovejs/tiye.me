记得例子不少用 guile 来运行 scm 的, 在脚本开头加两行并可以空行
	#! /usr/bin/env guile
	!#
系统没有 guile 可以在 Ubuntu 安装, 我装的是 1.8 版本
然后脚本我不重复了, 开头缩进是笔记格式, 代码参考原文
	;The first program
	(begin
		(display "Hello")
		(newline))
分号开头进行注释, `begin`表示后边
